# Deepak Nikhil & Co. — Financial Architecture Platform

A modern, elegant web platform for Deepak Nikhil & Co., showcasing comprehensive financial services including corporate structuring, taxation, audit & assurance, strategic advisory, and more.

## 🌟 Features

- **Modern Design**: Responsive layout with smooth animations and interactive components
- **Services Showcase**: 11 comprehensive financial services displayed elegantly
- **WebGL Background**: Interactive 3D visual elements powered by Three.js
- **Smooth Scrolling**: Lenis integration for fluid scroll experiences
- **Dropdown Navigation**: Clean, minimalist menu with transparent styling
- **Leadership Section**: Partner showcase and team presentation
- **Accessibility**: Semantic HTML and ARIA labels for better accessibility

## 🛠️ Tech Stack

- **Frontend Framework**: Vite + TypeScript
- **Styling**: CSS3 with Tailwind utilities
- **3D Graphics**: Three.js
- **Animation**: GSAP (GreenSock Animation Platform)
- **Smooth Scrolling**: Lenis
- **Fonts**: Google Fonts (DM Sans, Playfair Display, Montserrat)

## 📦 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <your-repo-url>
cd "CA DeepakNikhil & CO"
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

5. Preview production build:
```bash
npm run preview
```
